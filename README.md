# Portfolio
## Student Academic Portfolio Website
### Overview
This project is a multi-page personal academic portfolio and student management website developed using HTML, CSS, and JavaScript.

## Features
- Homepage
- About Me page
- Projects page
- Academic Planner
- Contact Form with Validation
- Responsive Design
- Interactive JavaScript Task Manager

## Technologies Used
- HTML5
- CSS3
- JavaScript
